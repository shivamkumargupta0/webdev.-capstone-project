# Zaika Express – Futuristic Food Delivery Landing Page

Zaika Express is a modern, sleek, and futuristic-themed food delivery landing page built using **HTML and CSS only**. It provides a visually appealing UI with interactive hover effects, smooth scrolling navigation, responsive layout support, and clickable functional sections that guide users throughout the interface.

---

##  Theme & Design Concept

The project follows a **futuristic soft gradient UI theme** inspired by modern delivery applications. The interface uses glowing accent colors, smooth shadows, rounded UI components, and premium typography to create a polished and clean user experience.

**Theme Characteristics:**
- Futuristic gradient background
- Neon hover glow animations
- Rounded UI elements and soft shadows
- Smooth scroll-based navigation
- Interactive category and restaurant cards
- Clean typography using Poppins font
- Fully static site with no JavaScript required

---

##  Features

| Feature | Description |
|--------|------------|
| Hero Section | Large featured image with main CTA button |
| Search Bar | location + category filter search component |
| Categories Grid | Clickable category cards with hover animations |
| Restaurants Section | Responsive restaurant cards with ratings & hover scaling |
| Promo Offers | Attractive promotional banners |
| Customer Reviews | Testimonials section with hover interaction |
| App Download | App store button UI section |
| Smooth Scroll Navigation | Top bar links scroll to specific page sections |
| Responsive Layout | Works across all devices using CSS grid & flex layout |

##  Responsive Breakpoints

The page is optimized for multiple screen widths:

| Device | Width |
|--------|-------|
| Extra Small Devices | ≤ 480px |
| Small Devices | 481px – 768px |
| Medium Devices | 769px – 1024px |
| Large Screens | ≥ 1025px |

The layout automatically adjusts using CSS Grid and Flexbox to maintain usability across all screen sizes.

##  How to Run / View the Website

### **Method 1: Open locally**
1. Download or clone the project files
2. Ensure `index.html` and `style.css` are in the same folder
3. Double-click `index.html` to open in any web browser

### **Method 2: Use VS Code with Live Server**
1. Install **Live Server extension**
2. Right-click `index.html`
3. Click **"Open with Live Server"**
4. Website will launch automatically in your browser

### **Method 3: Host on GitHub Pages**
1. Create a repository and upload project files
2. Go to **Settings → Pages**
3. Select **Source → main / root**
4. Live URL will be generated automatically
